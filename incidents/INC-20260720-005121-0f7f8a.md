# Сова создала инцидент, проигнорировав чеклист скила registering-incident

## Описание
Сова (owl, аудитор) при регистрации инцидента про задержку ответа Доминики (первичный файл `INC-20260720-004725-dmnrsp.md`) не использовала скил `registering-incident` и его чеклист. Файл создан напрямую через `write` с произвольным суффиксом `dmnrsp` вместо:
- генерации короткого hash-суффикса (Шаг 0 / формат скила);
- Bole-дедупа по `patterns/PAT-019-lab-instincts-library.md` (Шаг 0) — пропущен, хотя это рецидив I-12;
- обязательной секции `## Паттерн (инстинкт)`;

- коммита через `lab-commit.sh`.
Имя файла также нарушено: использован префикс `INC-` в имени (скил требует `YYYY-MM-DD-HHMMSS-<hash>.md` без префикса), суффикс не случайный.

## Решение
(заполняется при закрытии — verifier: ЗавЛаб, ADR-0057)
