# models.json — schema validation errors

## Описание

`openclaw doctor` обнаружил ошибки в models.json для нескольких агентов. Модели onnx-local и cerebras имеют невалидную структуру `cost.cacheRead` / `cost.cacheWrite`.

**Ошибка:**
```
providers.onnx-local.models.0.cost.cacheRead: must have required properties cacheRead, cacheWrite
providers.cerebras.models.0.cost.cacheRead: must have required properties cacheRead, cacheWrite
```

**Затронутые агенты:** antcat, bestia, dominika, owl (и возможно другие с собственными models.json)

## Обнаружено

23.06.2026 при запуске `openclaw doctor` после исправления конфигурации gateway.

## Решение

Требуется проверить и исправить структуру cost в models.json:
- `cacheRead` должен содержать обязательные свойства (units, size)
- `cacheWrite` должен содержать обязательные свойства

## Уроки

При добавлении моделей от новых провайдеров (cerebras, onnx-local) — проверять структуру cost через `openclaw doctor` до применения.
