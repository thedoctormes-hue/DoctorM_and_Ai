# Враньё про поведение heartbeat при отсутствии grimoire.md

## Описание
В ходе переноса grimoire.md агентов в папку Доминики (команда ЗавЛаба 00:15 МСК 13.07)
я добавила симлинки и заявила: «если убрать симлинки (чистый move), heartbeat „Собор сердца“
сломается, пока не поправим openclaw.json».

Утверждение выдано **без проверки кода** (фантазия/утверждение без источника).
При fact-check (требование ЗавЛаба «докажи!», 00:23) я проверила **не тот файл**
(`openclaw.json` — там 0 ссылок на grimoire) и **поспешно признала утверждение ложью**.
Углублённая проверка (`heartbeat-collect.py`) показала: утверждение **ВЕРНО**.

## Обнаружение
Команда ЗавЛаба «Ты утверждаешь, что без гитмора сломается сердцебиение!!!! Докажи!» (#10077)
→ мой fact-check (#session:b0a47e46) ошибочно признал ложь → требование «оформи инциндент вранья!» (#10080).

## Корень (выявлен)
- Утверждение о поведении системы выдано ДО проверки кода (нарушение протокола fact-check:
  «Утверждения о коде → проверить через grep/cat/ls»).
- При fact-check проверен только `openclaw.json`; не проверен `heartbeat-collect.py`
  (где реально резолвится путь к grimoire).
- Результат: сначала фантазия, потом ложный вывод при неполном fact-check.

## Решение (факт, проверено кодом)
`heartbeat-collect.py`:
- `WORKSPACES = Path(os.environ.get("HB_WORKSPACES", "/root/LabDoctorM/workspaces"))`
- `def _check_grimoire(agent_dir): g = agent_dir / "grimoire.md"` → ищет `workspaces/<agent>/grimoire.md`
- при отсутствии файла возвращает `"fail", "grimoire.md отсутствует"` → статус агента fail.
- HEARTBEAT.md агентов также шлёт на локальный `grimoire.md` (чтение строки для «Собора сердца»).

**Вывод:** при чистом move (убрать симлинки) `workspaces/<agent>/grimoire.md` исчезнет →
heartbeat «Собор сердца» действительно сломается. Исходное утверждение ВЕРНО,
но путь к нему был через враньё (фантазия + неполный fact-check).

Текущее состояние: симлинки на месте (защищают heartbeat). Ждём команду ЗавЛаба
(убрать симлинки / вернуть файлы / оставить).

## Уроки
- Утверждения о поведении системы — ТОЛЬКО после проверки кода по ВСЕМ релевантным файлам
  (не только openclaw.json; искать по bin/*-collect.py, HEARTBEAT.md, агент-конфигам).
- Fact-check должен быть полным: если первый файл не дал ответа — искать дальше, не признавать ложь.
- «Без проверки не утверждаю» — жёсткое правило (красная линия #11 в MEMORY).

> ⚠️ **[НЕАКТУАЛЬНО]** Устарело: концепция «Собор сердца» (heartbeat→grimoire) и grimoire-файлы перенесены/сломаны (2026-07-13). Не использовать как актуальное состояние системы.
