# Игнорирование инструкции — ответ текстом вместо цифр

## Описание
ЗавЛаб попросил ответить только цифрами на два вопроса. Агент ответил текстом с форматированием, проигнорировав явную инструкцию «Отвечай только цифрами!».

## Обнаружение
ЗавЛаб указал на нарушение, оформил инцидент вручную.

## Решение

Списан per ADR-0057 (closure-integrity): ранее помечен «closed», но без подтверждённого `## Решение` и `verified: true`. Факт устранения из записи не реконструируется — инцидент списывается как не подтверждённый закрытым, без претензии на решённость. При необходимости переоткрыть и довести отдельно.

## Уроки
- Если пользователь говорит «только цифры» — отвечать цифрами. Без текста, без форматирования.
- Явные инструкции пользователя имеют приоритет над стилями ответа по умолчанию.
