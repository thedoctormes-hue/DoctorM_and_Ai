# INC-028: Регистрация manus-outsourcing в skills.entries

**Проблема:** Скил `manus-outsourcing` существовал на диске (`/root/.openclaw/skills/manus-outsourcing/SKILL.md`), но не был зарегистрирован в `openclaw.json`. ЗавЛаб не знал что скил "просто положить в папку" достаточно для кастомных скилов.

**Решение:**
1. Добавлена запись `"manus-outsourcing": {"enabled": true, "env": {}}` в `skills.entries`
2. `openclaw config validate` → OK
3. `openclaw doctor` → OK
4. Бэкап: `openclaw.json.bak.2026-06-25T17-30-17`

**Урок:** Кастомные скилы в `/root/.openclaw/skills/` сканируются автоматически. Запись в `skills.entries` нужна только для контроля `enabled/disabled` и передачи `env`.

**Решение:** ✅ зарегистрирован
