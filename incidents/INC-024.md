# INC-024: WorkspaceVanishedError — массовый сбой

## Описание
После миграции путей воркспейсов Вороном (коммит `6d0ab797`, 00:35 UTC) все 8 агентов получили `WorkspaceVanishedError`. 40020 ошибок за ~30 минут.

## Корневая причина
1. Ворон мигрировал `DoctorM_and_Ai` → `LabDoctorM` (40K+ файлов)
2. Конфиг обновился, но gateway кэшировал старые пути сессий
3. Spooled updates с ошибками → бесконечный retry

## Решение
- Рестарт gateway (01:23 UTC)
- Ошибки прекратились (последняя 01:27 UTC)
- Все воркспейсы в `/root/LabDoctorM/workspaces/` на месте

## Урок
Миграция путей воркспейсов — только в maintenance window с рестартом gateway.
