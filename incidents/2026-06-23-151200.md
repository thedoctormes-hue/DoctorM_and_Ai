# Коллапс cron-задач — 5 из 22 задач падают по cascade failure

## Описание

5 из 22 cron-задач имеют consecutiveErrors > 0. Корневой анализ выявил системную проблему: неправильные параметры конфигурации gateway приводят к каскадному падению ВСЕХ cron-задач.

**Проблемные задачи:**
- compaction-quick-check: 4 consecutive errors, timeout 60s
- dominika-cve-scan: 4 consecutive errors, timeout 120s
- Memory Dreaming Promotion: 5 consecutive errors, timeout 636s
- mangust-adr-scan: 1 error, delivery accountId="mangust" (нет бота)
- filter-free-models-12h: 1 error, gateway restart

**Корневые причины (подтверждены документацией + логи):**

1. **maxConcurrent=3** — слишком мало для 9 агентов + 22 cron-задач. Очереди по 56+ секунд (`lane wait exceeded: waitedMs=56768`)
2. **contextTokens не задан** — дефолт 200K при реальном окне owl-alpha 1.0M. Приводит к context overflow на длинных сессиях (dominika: 376 сообщений = ~273K токенов = 136% от лимита)
3. **diagnostics не настроены** — stalled sessions (antcat 286s, dominika 271s, kotolizator 172s) висят бесконечно с `recovery=none`, блокируя слоты
4. **compaction.timeoutSeconds=120** — мало при очередях, приводит к cascade failure: overflow → compaction timeout → сессия зависает → ещё больше очередь
5. **trajectory файлы по 10MB** (суммарно 1.2GB) — bootstrap overhead при каждом старте сессии
6. **toolsAllow не оптимизирован** — cron-задачи используют 27 инструментов при нужных 2-4

## Решение

Применено. Ожидаемая верификация: следующие запуски cron-задач должны пройти успешно.

## Уроки

1. **maxConcurrent должен учитывать количество агентов + cron-задач.** Формула: агенты × 1.5 + cron_heavy × 0.5
2. **contextTokens должен соответствовать реальному окну модели.** Заниженный кап = context overflow = compaction cascade
3. **diagnostics должен быть настроен явно.** Без stuckSessionAbortMs stalled sessions висят бесконечно
4. **toolsAllow в cron-задачах должен быть минимальным.** 27 инструментов = ~6K токенов bootstrap overhead
5. **delivery accountId должен указывать на существующий аккаунт с ботом**
