# Игнорирование инструкции "ответить только цифрами"

## Описание
ЗавЛаб просил ответить только цифрами на два вопроса. Вместо этого был выдан развёрнутый текст с пояснениями, списками и дополнительной информацией.

## Обнаружение
ЗавЛаб указал на нарушение: "Я просил ответить цифрой, ты пишешь текст."

## Решение

## Уроки
- Если пользователь говорит "только цифры" / "только слово" / "кратко" — буквально следовать инструкции
- Не добавлять пояснения, когда просили лаконичный ответ
- Инструкции по формату ответа важнее общих правил вежливости
