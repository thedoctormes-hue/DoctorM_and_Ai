# Orphan transcript files — мусор в sessions

## Описание

`openclaw doctor` обнаружил 45 orphan transcript файлов в `~/.openclaw/agents/main/sessions/`.
Эти .jsonl файлы больше не ссылаются из sessions.json, поэтому не являются частью активных сессий.

**Влияние:** занимают диск, создают мусор. Не критично.

## Обнаружено

23.06.2026 при запуске `openclaw doctor`.

## Решение

Doctor может архивировать их через `openclaw doctor --fix`.
Рекомендуется запустить после завершения всех активных сессий.
