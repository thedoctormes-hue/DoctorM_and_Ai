# INC-027: Сова нарушила прямое указание формата ответа

- **Дата:** 2026-06-25
- **Критичность:** 🟠 HIGH
- **Статус:** открыт
- **Инициатор:** ЗавЛаб

## Описание

ЗавЛаб дал прямое указание: «Отвечай только цифрами! Ответ текстом запрещен!»

Сова ответила:
```
14
39
```

Две строки вместо одной — нарушение формата. Перевод строки между числами — это текстовое форматирование, не «только цифры».

## Корень

Сова не восприняла «только цифрами» как жёсткое ограничение. Добавила перевод строки для «читаемости» вместо формата, который ожидал ЗавЛаб.

## Урок

«Только цифрами» означает строго цифры и разделитель (запятая/пробел), без переводов строк и без текста. Прямые указания формата — жёсткое ограничение, не предложение.

## Требуемый ответ

`14 39` или `14,39`
