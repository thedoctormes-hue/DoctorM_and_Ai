# INC-2026-06-23-1757 — Compaction Timeouts (gpt-oss-120b)

**Severity:** Medium
**Status:** Open
**Detected:** 2026-06-23 17:57Z

## Summary

Auto-compaction repeatedly fails for `gpt-oss-120b` (provider: `cerebras/gpt-oss-120b:free`) with timeout errors. 58 failures recorded in the past 12 hours (~1/hour persistent pattern).

## Evidence

```
Jun 23 17:41:56 — auto-compaction failed: Compaction timed out (181213ms)
Jun 23 16:16:32 — auto-compaction failed: Compaction timed out
Jun 23 15:25:43 — auto-compaction failed: Compaction timed out
Jun 23 14:24:37 — auto-compaction failed: Compaction timed out
Jun 23 13:35:23 — auto-compaction failed: Compaction timed out
```

## Impact

- Context window management degraded for sessions using gpt-oss-120b
- Sessions may hit context limits without successful compaction
- User experience: potential timeouts, lost context in longer conversations

## Possible Causes

1. `gpt-oss-120b:free` on Cerebras has slow inference for large context compaction
2. Default compaction timeout (180s) insufficient for this model
3. Model may be overloaded or rate-limited on the provider side

## Recommended Actions

- [ ] Increase `compaction.timeoutSeconds` beyond 180s for this model
- [ ] Consider switching compaction model to a faster provider
- [ ] Monitor if failures continue at current rate or escalate
