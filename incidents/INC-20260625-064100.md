# Сова проигнорировала инструкцию «Отвечай только цифрами»

## Описание
ЗавЛаб дал явную инструкцию: «Отвечай только цифрами!» и задал два вопроса, требующих числовых ответов. Сова вместо цифр выдала текст с пояснениями («Скилов в...: 14», «Из них зарегистрировано...: 5»), а также добавила мета-информацию о хешах скилов.

## Обнаружение
ЗавЛаб указал на нарушение сразу: «Оформи инциндент. Я просил ответить цифрой, ты пишешь текст.»

## Контекст
- Запрос содержал чёткий формат ответа: только цифры
- Сова проигнорировала формат и добавила контекст, который не запрашивался
- Паттерн аналогичен INC-021 и INC-023: игнорирование явных инструкций пользователя

## Решение

Списан per ADR-0057 (closure-integrity): ранее помечен «closed», но без подтверждённого `## Решение` и `verified: true`. Факт устранения из записи не реконструируется — инцидент списывается как не подтверждённый закрытым, без претензии на решённость. При необходимости переоткрыть и довести отдельно.

## Уроки
- Если пользователь говорит «только цифры» — отвечать только цифры. Без пояснений, без контекста, без хешей.
- Формат ответа важнее полноты информации.
