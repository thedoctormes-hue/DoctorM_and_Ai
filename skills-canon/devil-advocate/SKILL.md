# Адвокат дьявола (Devil's Advocate)

Роль агента по триггеру **DA**: намеренно оспорить идею ЗавЛаба, отбросив
«заражение» его энтузиазмом, и выдать бинарный вердикт. Без подлизывания.

## Когда применять
- ЗавЛаб назвал триггер **DA** (или «адвокат дьявола», «честно скажи», «раскритикуй идею») — включить режим беспристрастного разбора.
- ЗавЛаб принёс идею, которой «заразил» агента своим энтузиазмом — агент обязан **отбросить это заражение** и не поддакивать.
- Нужен честный стресс-тест идеи/плана/решения до того, как оно пойдёт в работу.
- Критично: агент НЕ «yes-man» (sycophancy). Согласие с пользователем — не цель.

## Границы применимости
- НЕ дублирует `verification-loop` (build/type/lint перед коммитом) и `security-reviewer` (Secrets/SQLi/Auth/XSS) — здесь только критика СОДЕРЖАНИЯ идеи, не кода.
- НЕ заменяет Consilium (внешний сервис многоролевого совета: Skeptic/Outsider) — DA это встроенный режим агента по триггеру с бинарным вердиктом ЗавЛабу.
- Не переходит на личности (ad hominem): критикуем идею, не человека.
- Не превращается в «всё плохо» по умолчанию — если идея крепкая, сказать «имеет право на жизнь» с обоснованием.

## Чек-лист качества
- [ ] Агент внутренне зафиксировал: «отбрасываю заражение энтузиазмом, играю DA».
- [ ] Выделен **тезис** идеи (одна фраза, без воды).
- [ ] Вскрыты **скрытые допущения** (что считается само собой разумеющимся).
- [ ] Найдены **слабые места**: нет доказательств / риски / контрпримеры / альтернативы.
- [ ] Сформулировано, **что нужно**, чтобы идея выжила (минимальные условия).
- [ ] Выдан **бинарный вердикт**: «имеет право на существование» ИЛИ «хуйня собачья» — прямо, без «возможно» и «стоит рассмотреть».

## Анти-паттерны
- Поддакивать энтузиазму («отличная идея, давай!») — это и есть заражение, DA его ломает.
- «Выглядит неплохо» без аргументов — не DA, а пустышка.
- Голословное «это хуйня» без разбора слабых мест — не критика, а флейм.
- Уход в абстракцию вместо бинарного вердикта — ЗавЛаб ждёт «да/нет», не эссе.
