# RUL-010: Для семантического поиска агентам разрешён ТОЛЬКО MCP memory-gateway__search_memory

## Принцип
Единственный разрешённый путь семантического поиска по лаборатории для любого агента —
MCP-инструмент `memory-gateway__search_memory`. Всё остальное запрещено.

Это **ЗАКОН** лаборатории — установлен ЗавЛабом 2026-07-16 на базе утверждённого
Канонического стека семантической памяти (БЛОК 2 = НЕ канон, запрещён).

## Категория
process / access

## Описание
- **Разрешено:** `memory-gateway__search_memory` (MCP, гибрид vector+lexical, RRF, context
  expansion). Единственный живой путь.
- **Запрещено агентам использовать для семантического поиска:**
  - `scripts/lab_search.py` — мёртв.
  - `onnx-embedder.service` :8082 (весь старый FAISS+ONNX пайплайн) — мёртв, зона antcat.
  - нативный OpenClaw `memory_search` — paused.
  - `mcp-memory` :8087 — удалён.
  - Context API :8100 — закрыт.
  - прямые вызовы ALM :3002 (anythingllm) на чтение/переэмбеддинг — запрещены
    (управление = зона Штрейкбрехера, RUL-009).

## Обязательно
- [ ] Агент для семпоиска вызывает ТОЛЬКО `memory-gateway__search_memory`.
- [ ] Любой другой путь (lab_search.py, onnx-embedder, native memory_search, mcp-memory,
      Context API, прямой ALM :3002) — ЗАПРЕЩЁН.

## Исключения
Только прямое письменное поручение ЗавЛаба конкретному агенту выполнить конкретный
поиск иным путём (не применяется к мёртвым/удалённым сервисам).

## Наказание за нарушение
Инцидент severity: high (как `INC-2026-07-16-000300`, `INC-2026-07-16-000301`).
Прямой вызов запрещённого пути = нарушение ЗАКОНА + вторжение в зону владельца (ADR-014).

## Связанные артефакты
- `RUL-009` — каноничным стеком управляет ТОЛЬКО Штрейкбрехер.
- `INC-2026-07-16-000300` — вторжение raven в legacy ALM :3002.
- `INC-2026-07-16-000301` — antcat самовольный reindex.
- Канонический стек семантической памяти (16.07.2026), БЛОК 1 (канон) / БЛОК 2 (не канон).
