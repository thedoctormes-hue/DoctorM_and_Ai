---# RUL-002: Проверка границ при удалении кода

## Категория
coding-style

## Описание
При удалении кода необходимо проверять границы секций. Каждая секция — отдельный блок. Нельзя удалять всё подряд.

## Обязательно
- [ ] Перед удалением определить точные границы секции (начало/конец)
- [ ] Проверить что соседние секции не затронуты
- [ ] Запустить тесты после удаления
- [ ] При удалении >50 строк — обязательный code review

## Исключения
Нет.

## Наказание за нарушение
**error** — откат изменений, восстановление из резервной копии.
