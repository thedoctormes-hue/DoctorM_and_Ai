# Ловец инсайтов

Это общий блок для всех линз. Каждая линза ссылается на этот файл вместо копипасты.

## Правило

При работе с любой линзой фиксируй найденные инсайты:

- `[INSIGHT: тип]` — краткое описание (что нашли, почему важно, что делать)

## Типы инсайтов

- `pattern` — найден повторяющийся паттерн
- `anomaly` — аномалия, отклонение от нормы
- `risk` — риск, который нужно зафиксировать
- `opportunity` — возможность для улучшения
- `debt` — технический долг

## Формат

```
[INSIGHT: pattern] Описание паттерна. Почему важно. Что делать.
[INSIGHT: risk] Описание риска. Почему важно. Что делать.
```

## Применение

Инсайты записываются в граф инсайтов лаборатории при завершении задачи.
