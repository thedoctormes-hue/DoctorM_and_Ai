# Линзы лаборатории

**Линза = думай так.** Скилл говорит «делай шаг 1, 2, 3». Линза говорит «задай себе эти вопросы».

Линзы — это не чек-листы. Это mental models, которые меняют как ты смотришь на задачу.

**Принцип:** линзы редкие и точные. Когда всё — линза, ничто — не линза.
