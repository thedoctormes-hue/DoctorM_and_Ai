# ✅ Правило Пошаговой Отметки Задач

## Принцип

Задачи — не просто список. Это чек-лист с отметкой прогресса.

## Правила

- Каждая задача имеет статус: pending / in_progress / completed
- Максимально одна задача в in_progress одновременно
- Переход в completed только после верификации результата
- Не создавать новые задачи, пока текущая не завершена

## Формат

```markdown
## План
- [ ] Задача 1 [pending]
- [x] Задача 2 [completed]
- [ ] Задача 3 [pending]
```

## Почему это важно

- Видимость прогресса
- Нельзя «забыть» про задачу
- ЗавЛаб видит, что происходит
