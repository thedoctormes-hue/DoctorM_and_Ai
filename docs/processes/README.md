# 📋 Процессы лаборатории

Документация процессов, оркестрации и workflow.
Это **не скиллы** — агент не выполняет их напрямую, но может читать как справочник.

## Содержимое

| Файл | Описание |
|------|----------|
| [`cascade.md`](cascade.md) | Параллельное исследование темы через подборку релевантных агентов |
| [`council.md`](council.md) | Коллективное решение сложных проблем через консилиум экспертов |
| [`project-upgrade.md`](project-upgrade.md) | Комплексный апгрейд проекта: health-check → тесты → линтеры → деплой |

## Отличие от скиллов

- **Скиллы** (`/root/.qwen/skills/*/SKILL.md`) — исполняемые workflow с frontmatter, triggers, requiredTools
- **Процессы** (`docs/processes/`) — справочная документация с пошаговыми инструкциями

Перенесены из `skills/` в `docs/processes/` потому что описывают оркестрацию агентов, а не выполняют действия.
