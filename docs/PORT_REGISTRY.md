# PORT_REGISTRY.md — Реестр портов лаборатории

> ⚠️ **Правило:** Каждый сервис слушает на `127.0.0.1:<port>`. Доступ извне только через nginx proxy.
> При конфликте портов — уведомить ЗавЛаба и обновить этот файл.

## Активные сервисы

| Сервис | Порт | Хост | Владелец | Описание |
|--------|------|------|----------|----------|
| myrmex-control | 3000 | 127.0.0.1 ✅ | Муравей | Дашборд лаборатории (React + Express) |
| snablab | 8001 | 127.0.0.1 | Бестия | СнабЛаб API (FastAPI) |
| vpn-daemon | 8080 | 127.0.0.1 | Кот | VPN бот (aiogram) |
| screenshot-service | 8190 | 127.0.0.1 | Штрейкбрехер | Playwright screenshot API |
| saas-api | 8190 ⚠️ | 127.0.0.1 | ??? | OpenClawBox API (КОНФЛИКТ с screenshot-service!) |
| nginx | 80/443 | 0.0.0.0 | Бестия | Reverse proxy + SSL |
