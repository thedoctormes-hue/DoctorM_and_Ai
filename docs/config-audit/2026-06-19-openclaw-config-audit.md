## ПРИОРИТЕТ 1 — Критические (безопасность)

### R-1. Gateway bind и reverse proxy
**Статус:** ✅ Частично
**Текущее:** `gateway.bind: loopback`
**Рекомендация:** Если есть reverse proxy (nginx/caddy/traefik), нужно добавить `trustedProxies` — иначе проверки IP будут работать некорректно.
**Действие:** Проверить наличие reverse proxy. Если есть — настроить `gateway.trustedProxies`.

### R-2. Secrets audit
**Статус:** ❌ Требует внимания
**Текущее:** Секция `secrets` отсутствует в конфиге.
**Рекомендация:** Прогнать `openclaw secrets audit --check` и найти открытые секреты. Мигрировать .env токены в SecretRef.
**Действие:**
```bash
openclaw secrets audit --check
```

### R-3. Auth token в открытом виде
**Статус:** ❌ Требует внимания
**Текущее:** `gateway.auth.token` лежит прямо в `openclaw.json`.
**Рекомендация:** Вынести в переменную окружения через SecretRef.
**Действие:** Создать SecretRef для auth token, обновить конфиг.

### R-4. Telegram группы
**Статус:** ✅ Заблокировано
**Текущее:** `groupPolicy: allowlist` с пустыми `groups`.
**Рекомендация:** Явно указать group IDs если планируется работа в группах. Или установить `groupPolicy: disabled` для явности.
**Действие:** Определить политику для групп.

## ПРИОРИТЕТ 3 — Желательные (качество работы)

### R-10. Распределить модели по агентам
**Статус:** ⚠️ Все на одной модели
**Текущее:** Все 9 агентов используют `default` (openrouter/owl-alpha).
**Рекомендация:** Для простых агентов (bestia, dominika) задать более лёгкую модель из fallback. Экономия токенов.
**Действие:** Определить приоритет агентов, назначить модели.

### R-11. memorySearch.provider
**Статус:** ⚠️ Не указан
**Текущее:** `provider: "none"`, но есть `remote` и `model: "bge-m3"`.
**Рекомендация:** Явно указать `provider: "ollama"` если Ollama доступен.
**Действие:** Проверить доступность Ollama, обновить конфиг.

### R-12. compaction.memoryFlush.model
**Статус:** ⚠️ Можно оптимизировать
**Текущее:** `nemotron-3-super-120b-a12b:free`
**Рекомендация:** Использовать самую быструю fallback-модель: `openrouter/openai/gpt-oss-20b:free`.
**Действие:** Изменить в конфиге.

### R-13. Агент ant vs antcat
**Статус:** ❓ Требует проверки
**Текущее:** В Telegram аккаунтах `antcat`, в списке агентов `antcat`. `ant` упоминался в INC-012 как удалённый.
**Рекомендация:** Убедиться что `antcat` — это и есть бывший `ant`, а не дубль.
**Действие:** Проверить историю, подтвердить отсутствие дублирования.

### R-14. Очистить неиспользуемые скиллы
**Статус:** ⚠️ 41 скилл
**Текущее:** Много macOS-специфичных скиллов (imsg, apple-notes, bear-notes, things-mac, sag).
**Рекомендация:** Оставить глобально только универсальные скиллы. Остальные — per-agent.
**Действие:** Проверить какие скиллы используются. Убрать неиспользуемые из глобального списка.
