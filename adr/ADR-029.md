# ADR-029: Vault Integration for Secret Storage
## Consolidated Insights

1. Токен в vault уже отозван (одноразовый). Ключ из предыдущего теста тоже не работает — видимо, ты его отозвал или он тоже был одноразовый.
    
    ЗавЛаб, вот что произошло:
    
    **Токен в vault одноразовый** — я пыталась получить его несколько раз, и он сработал только один раз (в первый раз я получила ключ `sk-proj-...`). Но я не сохранила его в `.env` сразу — пыталась через bash-подстановку, которая не сра...

2. Понял контекст. Сова получила одноразовый токен vault, тестировала API ключи (Cloudflare, Manus), заносила результаты в `model_registry.md`. Сессия завершилась нормально — она не устала, а доработала.
    
    Но ты говоришь «подхватишь её работу» — значит есть что доделать. Давай уточню:
    
    - Какой конкретно API ключ нужно дотестировать?
    - Или нужно продолжить каталогизацию моделей в `model_registry.md`?...
