# ADR-0046: Heartbeat-based Config Lint вместо cron job

**Дата:** 2026-06-20
**Статус:** Accepted
**Автор:** Штрейкбрехер (Streikbrecher)
**Связанные коммиты:** `21e6bc01`

## Контекст

Отдельный cron job `streikbrecher-config-lint` для проверки конфигурации был некорректно настроен и генерировал ложные срабатывания (GitHub issue #43850). Требуется более надёжный механизм проверки.

## Решение

Замена cron job на **heartbeat-based config lint**:

- Еженедельная проверка встроена в `HEARTBEAT.md` (воскресенье ~04:00 UTC)
- Тесты: `tests/test_doctor_lint.py` (6 тестов, все проходят)
- При обнаружении проблем — создаётся инцидент-отчёт

## Последствия

- Упрощение инфраструктуры: на один cron job меньше
- Проверка привязана к heartbeat-циклу агента (гарантированный запуск)
- Ложные срабатывания устранены

## Альтернативы

- Исправить cron job (отвергнуто: корневая проблема в конфигурации cron)
- GitHub Actions (отвергнуто: нет доступа к локальным файлам)
