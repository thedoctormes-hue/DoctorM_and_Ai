# ADR-017: В Myrmex Control можно добавить секцию Продуктивность — KPI

## Статус
proposed

## Контекст
В Myrmex Control можно добавить секцию Продуктивность — KPI для каждого лаборанта. Вместо отдельного скрипта glory_board.sh (который пишет в файл, который никто не читает), метрики продуктивности должны быть частью Myrmex Control. Интеграционная точка уже есть: POST /api/sessions/:id/complete вызывается при завершении каждой сессии. Нужно добавить сбор метрик (задачи, коммиты, инсайты) и агрегацию по лаборантам/неделям. Данные реальные: кто сколько закрыл задач, сколько коммитов, какие проекты двигает. Без рангов и скоров — просто факты для принятия решений.
