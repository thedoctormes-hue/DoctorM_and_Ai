# ADR-037: Agent Registry — openclaw.json как единственный источник истины

## Статус

Принято

## Контекст

В лаборатории используется файловая система для хранения информации об агентах (workspaces, skills, конфигурации). Возникает вопрос — где находится канонический реестр агентов?

## Решение

Для вопросов о реестре агентов единственный источник истины — `openclaw.json`.

- Все агенты регистрируются в `openclaw.json` → `agents.list`
- Каждый агент имеет уникальный `id`, `name`, `workspace`, `agentDir`
- Дублирование реестра в файловой системе запрещено
- Изменения реестра — только через редактирование `openclaw.json`

## Последствия

- Единый источник истины для всех агентов
- Нет рассинхронизации между файлами
- Упрощение поиска и навигации
