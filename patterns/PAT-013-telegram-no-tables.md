# PAT-013: Запрет использования таблиц в Telegram-сообщениях

## Контекст
Markdown-таблицы в Telegram ломают отображение: появляются вертикальные линии, текст выравнивается криво, на мобильных устройствах таблицы не читаемы.

## Решение
**Запрещено** использовать markdown-таблицы в Telegram-сообщениях. Вместо них использовать:

1. **Списки с отступами** для структурированных данных
2. **Блоки цитирования** (`>`) для выделения важного
3. **Эмодзи-маркеры** (🔴, 🟡, 🟢) для статусов
4. **Переносы строк** и пустые строки для разделения блоков

Пример правильного формата:
```
🔴 Проблема:
- Таблицы ломаются в Telegram
- Сложно читать на мобильных

🟡 Решение:
> Используйте списки с отступами
> Разделяйте блоки пустыми строками
> Добавляйте эмодзи для визуального выделения

🟢 Результат:
- Читаемо на всех устройствах
- Сохраняет структуру
```

## Следствия
- Читаемые сообщения в Telegram на всех устройствах
- Согласованный стиль коммуникации
- Уменьшение времени на deciphering форматов
